# 选题库 & 平台策略

这份文档是这档节目的"编剧本"。先讲**怎么发**，再给**20 期选题**。

---

## 一、平台策略

不同平台的内容形态差别很大，别用同一条视频硬发所有平台。核心是**一次创作，三种剪法**。

### 小红书（起号首选）
- **形态**：封面图（一张最惊艳的成品）+ 4–6 张过程图 + 30–60 秒竖屏视频
- **钩子**：标题走"反差感"——「我用 200 行代码画了一幅水墨」「文物 activated」
- **正文**：3 行以内讲清"这是什么 + 怎么做到的 + 附代码"，堆话题词 `#创意编程 #生成艺术 #国潮 #p5js`
- **为什么**：国潮 + 视觉 + "教程感"在小红书传播力最强，评论区天然会问"怎么做"

### B站（沉淀技术受众）
- **形态**：5–10 分钟横屏拆解视频，从成品倒推原理，最后开源代码
- **钩子**：前 15 秒必须放完整成品 + "这期教你用 XX 行代码做出它"
- **结构**：成品演示 → 拆一个核心算法（流场/递归/物理）→ 逐步加参数 → 交互演示 → GitHub 地址
- **为什么**：B站观众愿意看长、愿意 star 仓库，是你的"技术信任"沉淀池

### 抖音 / 快手（拉新流量）
- **形态**：竖屏 15–30 秒，**前 3 秒直接出最炸的那一下**（比如手指划过墨迹流动）
- **钩子**：无口播，卡点音乐 + 字幕「代码画的」「拖动试试」
- **为什么**：算法推荐吃"完播 + 互动"，交互作品天然让人想评论"这怎么做的"

### YouTube / X（可选，国际化）
- **形态**：英文，creative coding 社区（#p5js #generativeart #creativecoding）
- **钩子**：东方美学在海外是差异化优势，一条 GIF + CodePen 链接就能传播

### 发布节奏建议
- 起步阶段：**每周 1 期**，先把小红书 + B站两个阵地做扎实
- 每期沉淀：作品进 `episodes/`，仓库 README 更新索引，视频简介放 GitHub 链接形成闭环

## 二、20 期选题库

每期给了：**母题 / 视觉目标 / 核心技术 / 交互钩子**。难度 ⭐（新手友好）→ ⭐⭐⭐（要碰 shader/物理）。

### 第一辑 · 水墨书画（最适合 p5.js 起步）
1. **墨林 Ink Forest** ⭐ — 水墨飞白。Perlin 流场叠墨成山水。*交互：拖动毛笔注墨。*（✅ 已做，见 `episodes/01`）
2. **飞白 Dry Brush** ⭐ — 书法枯笔。粒子沿笔画路径流动，随机断续成飞白。*交互：写字，笔速控墨量。*
3. **泼墨 Splash** ⭐⭐ — 泼墨山水。反应扩散(reaction-diffusion)扩散墨团。*交互：点击泼一团。*
4. **印章 Seal** ⭐ — 篆刻朱文。网格 + 布尔运算生成印章边框与残破。*交互：输入名字生成印。*

### 第二辑 · 纹样器物
5. **饕餮 Taotie** ⭐⭐ — 青铜纹样。轴对称 + 递归回纹，一键生成一件"青铜器"。*交互：拖动改繁简。*
6. **青花 Blue-White** ⭐⭐ — 青花瓷开片。Voronoi 裂纹 + 缠枝纹填充。*交互：转动瓶身。*
7. **瓦当 Roof-tile** ⭐ — 汉代瓦当。极坐标 + 四分对称的云纹/文字。*交互：换字。*
8. **缂丝 Kesi** ⭐⭐ — 织物经纬。像素网格 + 织纹算法把图案"织"出来。*交互：上传图 → 织成缂丝。*
9. **榫卯 Joinery** ⭐⭐⭐ — 木构爆炸图。three.js 里榫卯咬合的拆解动画。*交互：拖动拼合。*

### 第三辑 · 建筑山水
10. **敦煌飞天 Apsaras** ⭐⭐ — 飘带。多段参数曲线 + 缎带几何，飘带随风。*交互：鼠标带风向。*
11. **千里江山 A Thousand Li** ⭐⭐ — 青绿山水。噪声地形 + 石青石绿分层上色，横向长卷。*交互：卷轴平移。*
12. **园林漏窗 Lattice** ⭐ — 花窗。递归分割生成万字/冰裂纹窗棂。*交互：换窗格样式。*
13. **斗拱 Dougong** ⭐⭐⭐ — 参数化斗拱。按等级生成不同复杂度的斗拱结构。
14. **星图 Star Map** ⭐⭐ — 苏州石刻天文图。真实星表 + 二十八宿连线。*交互：拖动看全天。*

### 第四辑 · 抽象 & 节气（国际化审美）
15. **二十四节气 Solar Terms** ⭐⭐ — 一期一节气，每个节气一套配色 + 生成规则（可做成一整年的连载）。
16. **流觞 Floating Cup** ⭐⭐⭐ — 曲水流觞。流体模拟(SPH 简化)水面漂杯。*交互：戳水面。*
17. **山海经 Bestiary** ⭐⭐ — 神兽。L-system + 器官拼装生成"异兽"。*交互：种子换一只。*
18. **围棋 Go** ⭐ — 棋局生长。落子扩散算法 + 黑白气韵。*交互：自己落子。*
19. **烟火 Fireworks** ⭐⭐ — 物理烟花。粒子系统 + 重力/拖拽 + 合成音效（呼应凤冠的 Web Audio）。*交互：点哪炸哪。*
20. **磬 Chime** ⭐⭐⭐ — 编钟。视觉波纹 + Web Audio 合成钟磬音，敲击可视化声音。*交互：敲不同的钟。*

## 三、复用性建议

- **第 15 期「二十四节气」**天生是连载引擎：一套模板 × 24 个变体 = 半年内容，且节气日发布蹭自然流量。
- **音效路线（19、20）**直接对标凤冠的 Web Audio 合成——"看得见的声音"在短视频里互动率极高。
- 每做完一期，把最惊艳的一张成品图存进 `assets/`，攒够 9 张就能发一条"我用代码画了 9 件文物"的合集，二次传播。
