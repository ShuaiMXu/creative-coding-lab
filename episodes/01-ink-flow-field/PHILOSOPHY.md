# 墨林 Ink Forest — 算法哲学

**母题**：中国水墨。墨分五色——焦、浓、重、淡、清。一笔下去，浓淡枯润皆在其中。

**核心比喻**：把每一道笔触当成一个**沿着"气韵"流动的粒子**。气韵是一张 Perlin 噪声场——看不见的力，却决定了墨往哪走。上千道笔触在同一张场里流淌、层层叠加，浓处成山，淡处成雾，断处成飞白。

## 算法怎么表达它

- **流场（flow field）**：整张画布是一个由 `noise(x, y)` 生成的角度场。每道笔触从随机点出发，每一步都读当前位置的噪声角度，朝那个方向走一小步，画一条极短的线。走上百步，就是一道蜿蜒的墨迹。
- **叠墨**：单步用很低的 alpha 描线，成百上千道叠在一起，自然形成浓淡过渡——这是水墨"晕"的来源，不是画出来的，是**叠出来的**。
- **飞白**：每一步以一定概率"跳过不画"，笔迹于是断续枯涩，得到枯笔飞白的质感。
- **点睛浓墨**：受种子控制的几处高斯散点，制造视觉焦点，避免整张平均。
- **可复现**：`randomSeed` + `noiseSeed` 锁定同一颗种子 → 永远同一幅画。这是生成艺术的"版号"，方便复现满意构图，也方便观众照着 seed 复刻。

## 交互

鼠标按住拖动 = 一支毛笔。笔尖顺着当前流场甩出一束墨，**笔速越快，墨越浓、越粗**。松手后这些墨已经落在纸上，成为画的一部分。这个"看得见的因果"是短视频里最抓人的一下。

## 参数（面板可调）

| 参数 | 作用 |
| --- | --- |
| 笔触密度 | 多少道墨迹——少则疏朗，多则厚重 |
| 墨迹长度 | 每道笔触流多远——短则点苔，长则山脉 |
| 气韵 Turbulence | 噪声尺度——小则舒缓长卷，大则湍急漩涡 |
| 墨色浓淡 | 单步 alpha——控制整体墨气轻重 |
| 飞白 | 断笔概率——0 则湿润，高则枯涩 |
| 纸色 | 宣纸 / 仿古 / 冷灰 / 墨底反白 |
