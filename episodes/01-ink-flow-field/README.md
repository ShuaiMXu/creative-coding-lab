# 第 01 期 · 墨林 Ink Forest（水墨流场）

> 用 Perlin 流场 + 叠墨，几百行 p5.js 画一幅会流动的水墨，还能用鼠标当毛笔往里注墨。

## 跑起来

双击 `index.html`，或本地起服务器：

```bash
python3 -m http.server 8000   # 然后打开 http://localhost:8000
```

右侧面板调参数，拖动画面注墨，点「存图」导出 PNG。算法原理见 [`PHILOSOPHY.md`](./PHILOSOPHY.md)。

## 出片脚本（这期怎么剪成视频）

**竖屏钩子（抖音/小红书，15–20 秒）**
1. `0–3s`：直接放一张已经画好的浓淡水墨成品（最惊艳那张，seed 记下来）。字幕：「这是代码画的」。
2. `3–8s`：手指/鼠标划过画面，墨迹顺着流场淌开——**这是最抓人的一下，一定要慢放**。
3. `8–15s`：快速拖动几下"泼墨"，画面越来越满。字幕：「拖动就能画」。
4. 结尾：一张最终成品定帧 + 「代码开源，主页拿」。

**横屏拆解（B站，5–8 分钟）**
1. 先放完整成品 + 交互演示（15 秒内）。
2. 讲"流场"是什么：一张看不见的风向图，粒子顺风走。现场把 `steps` 从 20 拉到 300，看墨迹变长。
3. 讲"叠墨"：把 alpha 调高一次看清单笔，再调低看叠出来的晕。
4. 讲"飞白"：飞白概率拉满，枯笔质感出现。
5. 上 GitHub，观众自己改。

## 可以怎么改（给观众留的作业）

- 换 `PAPERS` 配色 → 石青石绿就成青绿山水。
- `ang = n * TWO_PI * 2.2` 里的 `2.2` 调大 → 漩涡更狂，接近抽象。
- 把噪声换成图片亮度采样（`get()`）→ 让墨迹沿一张照片的轮廓流，变成"水墨滤镜"。
